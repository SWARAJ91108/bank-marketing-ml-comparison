# Bank Marketing Prediction using Machine Learning

## Problem Statement

Predict whether a bank customer will subscribe to a term deposit based on demographic information, financial details, and previous marketing campaign interactions.

## Algorithms Used

- Logistic Regression
- KNN
- Naive Bayes
- SVM
- Decision Tree
- Random Forest
- AdaBoost
- Gradient Boosting
- XGBoost
- Stacking

## Evaluation Metric
- Accuracy Score

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- XGBoost
- Matplotlib

## Results

Compared multiple supervised learning algorithms and visualized their performance using accuracy comparison charts.
